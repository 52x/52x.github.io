安装软件包

基本配置

执行pppoe-setup 选择对应网卡，配置DNS（可选），大部分默认即可，防火墙选择0，即不启用防火墙。

详细配置

  • /etc/ppp/options，如果没有此文件则创建，内容如下，注释login参数

  • /etc/ppp/pppoe-server-options，注释login参数，并使用chap认证。

  • /etc/ppp/chap-secrets，添加账号密码，可单独对某个账号指定IP。

运行PPPoE server

拨号

使用添加的账号拨号，即可看到分配到的IP为192.168.5.x

FAQ

  • 如果运行pppoe-server不成功，或一拨号就挂掉。
  • 估计是配置不正确，建议删除/etc/ppp/文件夹，重新安装最新版rp-pppoe。
  • 出现拨号651。
  • 产生原因有：
    • 没有/etc/ppp/pppoe-server-options文件。
    • /etc/ppp/pppoe-server-options文件没有auth和require-chap参数。
    • windows拨号时指定了服务器，要么去掉指定的服务器名称，要么pppoe-server添加-S参数指定服务器名称。
